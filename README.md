# Project
Bank Project
# Brief Description
This project applies machine learning techniques that were used in a Bank Marketing dataset in order to predict whether a client will subscribe to a term deposit. The dataset includes 45,211 observations and 17 features, which the code created first analyzed the dataset to understand key patterns. The data was also analyzed by using categorical variables and split into training and testing sets (80/20). The project offers a wide variety of ways to analyze and visualize the dataset as well as including exploratory data analysis (EDA) and performance evaluation. The goal was to identify the patterns in customer behaviors and build a model that will help in predicting if a client will subscribe to a term deposit or not.
